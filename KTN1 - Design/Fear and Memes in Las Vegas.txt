/ - denotes serverside action

CONNECTION SETUP:

User connects to Server
--> Requests to set up a connection
/Server recieves and accepts connection

LOGIN:

User attempt to log in:
IF(ANYTHING ELSE) -> Returns Error Message.
IF(YES) ->
/Server verifies unique valid username.  
IF(NO) -> Returns Error Message
IF(YES) ->  
/Server parses and sends messageLog to client.
Client is now able to recieve and post messages.


SEND MESSAGE:

User sends message
/Server (in message reciever) recieves message
/Server (sends to message parser).
/Server adds message to messageLog
/Server sends to all connected clients.
